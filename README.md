# Advanced-Calculator
Create Advanced Calculator Using Python Language

# How To Work
Select any option for calculation and you can add two number after choosing selection.

# Select operation.
1.Add      : + <br>
2.Subtract : - <br>
3.Multiply : * <br>
4.Divide   : / <br>
5.Power    : ^ <br>
6.Remainder: % <br>
7.Terminate: # <br>
8.Reset    : $ <br>
8.History  : ? <br>
